Contribution

I have worked on adding standard closed captioning CEA 708 features to VLC in addition to existing EIA 608 features.

Appearance

The additional structures are added for new features added eia708_PenAttribs_Size - 3 text sizes

eia708_PenAttribs_FontTag - 8 font options

eia708_PenColor - 64 text colors, and 64 background colors

eia708_PenColor_Opacity - background opacity and dropshadowed or edged text and other visual features

Language

708 captions have a complete library of special characters and symbols and are designed to support any alphabet in the world

Position

708 captions do not have to be fixed in one position. The window position can be adjusted accordingly which makes the user easy to change it according to the video which is being watched. eia708_WindowAttribs,eia708_WindowAttribs_AnchorID,eia708_WindowAttribs_Justify are the new structures defined to define the position.

The above structures are defined with the knowledge from the wikipedia page and some other sources. Some of basic set functions are also defined. pen color,pen attributes,pen location are all set to default. The reset function is also added.

The main structure eia708_t which is defined which consists of n windows.The present window has the features of pen,position and data.

Compiling VLC - Basics

The basics of compiling VLC in debian/ubuntu are already given in the VLC Compile The below mentioned are some of the problems i've faced and solutions to them.I've also put some basics of software programming.

I have put all the basic dependancies required. Please go through how to setup github

Environment:

VLC requires a C99 compiler, development headers and a toolchain. If you build from the Git repositories, you will also need the GNU build system, a.k.a. the "autotools" (autoconf, automake, libtool and gettext) to setup the Makefiles. Make sure they are up-to-date and usable for your system. 

$ sudo apt-get install git libtool build-essential pkg-config autoconf

Get the source :

$ git clone git://git.videolan.org/vlc.git
$ cd vlc
$ ./bootstrap

"Bootstrapping" is used as a term for building a system using itself -- or more correctly, a predecessor version. In computing, a bootstrap loader is the first piece of code that runs when a machine starts, and is responsible for loading the rest of the operating system. In modern computers it's stored in ROM.

Libraries and dependencies:

$ sudo apt-get install libjack-jackd2-dev
